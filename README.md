# Google Junior Web Portfolio

A simple static portfolio website ready for GitHub upload and GitHub Pages deployment.

## Files
- `index.html`
- `styles.css`

## How to upload to GitHub
1. Create a new repository on GitHub.
2. Upload all files from this folder.
3. Commit the files.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, set source to **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Save and wait for GitHub Pages to publish the site.

## What to customize
- Replace `your@email.com` with your real email
- Replace GitHub and LinkedIn placeholders
- Add real project links for each project card
- Swap artwork placeholders with screenshots of your work
- Update the name if needed
